Sistema de Clasificación de Pedidos – Entrega 2

## Descripción

Este proyecto es un programa en consola hecho en C# que simula el sistema de pedidos de una tienda online.
A diferencia de la Entrega 1 (que solo procesaba un pedido), en esta versión el sistema permite registrar varios pedidos durante la ejecución y luego generar un reporte con la información acumulada.

---

## 🎯 Objetivo

Aplicar estructuras vistas en clase como:

do-while
switch
List<T>


El sistema tiene un menú:

1. Registrar pedido
2. Ver reporte
3. Salir

---

##  Nota

* Los datos no se guardan permanentemente (solo mientras corre el programa)
* Solo se manejan valores positivos
